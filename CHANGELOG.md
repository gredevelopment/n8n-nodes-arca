# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.1] - 2026-04-14

### Added

- Initial release of Arca n8n community node
- Support for Arca API resources:
  - User operations
  - Workspace operations
  - Folder operations
  - List operations
  - Task operations
  - Status operations
  - Label operations
  - Comment operations
- Arca Trigger node for webhook-based automation

## [1.0.0] - 2026-04-14

### Added

- Initial release
- Complete Arca API integration
- Support for all major Arca resources
- Webhook trigger support
